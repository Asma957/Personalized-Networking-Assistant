# Brainstorming & Ideation

## Project Idea
Personalized Networking Assistant — AI-powered conversation starter generator for networking events.

## Problem Statement
Professionals struggle to initiate meaningful conversations at networking events.

## Proposed Solution
- Use AI (Gemini) to generate personalized conversation starters
- Extract themes from event descriptions
- Fact-check using Wikipedia API
- Save conversation history for future reference

## Key Features Identified
1. Smart conversation starter generation
2. Event theme analysis
3. Wikipedia fact verification
4. Conversation history & feedback system
